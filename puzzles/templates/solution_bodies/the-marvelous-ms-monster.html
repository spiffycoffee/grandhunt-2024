Each clip has a number of common components: a thumbnail, a clip with a sound effect pause in the middle, a “next video” thumbnail at the end that wraps around to the first clip, and a number of likes 2. Overlay the “next video” thumbnail over the actual next video's thumbnail. The red box will highlight a specific word in each clip, resulting in the clue AFTER SOUND 10TH WORD LATEST VIDEO 3. Each sound effect pauses the clip momentarily. Take the 10th word after each sound effect: 70K, AGES, BUNKER, YACHT, FBI, TRIANGLE, JET, ESCAPE 4. Either from clues in the flavor text, MsMonster's name itself, or from the first letter of each sound effect in the transcript, identify the tie to MrBeast, one of the biggest YouTube influencers. Each word identified matches a word in one of his YouTube video's titles. If there is more than one video that has that word, take the latest one (that existed before the upload date February 10th 2024) 5. As hinted in the flavor text, convert the number of likes in each MsMonster clip to seconds, or a “mm:ss" timestamp. In each MrBeast video, scrub to that timestamp 6. Overlay the same “next video” thumbnail on top of the image of each MrBeast video at each timestamp. Each overlay is highlighting the shape of a letter, resulting in the final answer {% endblock %} {% block author-notes-html %} This one was a weird one.
